# RxChat
Chat app made by RxSwift
RxSwift와 MVVM패턴을 익히기 위해 진행한 개인 프로젝트.

## 활용기술
- Swift
- RxSwift
- MVVM 패턴
- UIKit
- Realm
- Firebase를 통한 BackEnd 구현
- Google SignIn을 통한 로그인 및 사용자 관리
